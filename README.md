# HIKARI PropTech Starter

<p align="center">
  <strong>Template de démarrage SaaS PropTech</strong>
</p>

## Stack

- React 18 + Vite
- Tailwind CSS 3
- Base44 (BaaS: auth, database, hosting)
- Stripe (subscriptions)
- Lucide React (icons)

## Démarrage

```bash
npx degit HIKARI-GROUP/hikari-proptech-starter my-proptech
cd my-proptech
npm install
npm run dev
```

## Structure

```
src/
├── pages/          # Pages React
├── components/      # Composants UI
├── lib/            # Utils & hooks
├── api/            # Client API
└── index.css       # Design tokens
```

## Fonctionnalités incluses

- ✅ Auth (login, register, OTP, Google OAuth)
- ✅ Dashboard layout
- ✅ Stripe checkout (subscriptions)
- ✅ Design system (dark theme)
- ✅ SEO (meta tags, sitemap)
- ✅ Responsive (mobile + desktop)
