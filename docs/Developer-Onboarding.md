# Developer Onboarding

> Welcome to HIKARI GROUP! 🎉

## Prerequisites

- Node.js 20+, npm or pnpm, Git, GitHub account, Base44 account

## Setup

```bash
git clone https://github.com/HIKARI-GROUP/<repo>.git
cd <repo>
npm install
npm run dev
npm test
```

## First Contribution

1. Read [CONTRIBUTING.md](../CONTRIBUTING.md)
2. Look for `good first issue` labels
3. Fork → Branch → Code → PR
4. Ask questions in [Discussions](https://github.com/HIKARI-GROUP/.github/discussions)
